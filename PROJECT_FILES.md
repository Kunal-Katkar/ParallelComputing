# Project Files Structure

## Essential Files for Running the Project

### Root Directory
- **Mathematics Domain (MPI)**:
  - `PiEstimation.cpp` - Pi estimation using dartboard method
  - `PiEstimation2.cpp` - Pi estimation using quadrant integration
  - `Integration_Any.cpp` - Integration of high-power functions
  - `prng_engine.hpp` - Parallel random number generator (used by math programs)
  
- **Finance Domain (OpenMP)**:
  - `normsinv.h` - Inverse normal CDF (used by Stage 1)
  - `StockMovementSimulation/` - Stock simulation directory

- **Documentation**:
  - `README.md` - Main project README
  - `PROJECT_EXPLANATION.md` - Detailed project explanation
  - `ARCHITECTURE_DIAGRAM.md` - Architecture diagrams
  
- **Build Scripts**:
  - `build_and_run.bat` - Build script for stock simulation

### StockMovementSimulation/
- **Source Files**:
  - `BrownianStockMovement1.c` - Stage 1: Random number generation
  - `BrownianStockMovement2.c` - Stage 2: Prefix sum computation
  - `BrownianStockMovement3.c` - Stage 3: Stock price calculation
  - `Simulation.c` - Visualization (optional, requires OpenGL)

- **Executables** (compiled programs):
  - `BrownianStockMovement1.exe` - Stage 1 executable
  - `BrownianStockMovement2.exe` - Stage 2 executable
  - `BrownianStockMovement3.exe` - Stage 3 executable

- **Output Files** (generated when running):
  - `RNG.txt` - Generated by Stage 1
  - `RNG2.txt` - Generated by Stage 2
  - `StockPrice.txt` - Generated by Stage 3

- **Documentation**:
  - `README.md` - Stock simulation README

## How to Run

### Stock Simulation:
1. `BrownianStockMovement1.exe` → Enter length
2. `BrownianStockMovement2.exe` → Automatic (reads RNG.txt)
3. `BrownianStockMovement3.exe` → Enter drift, volatility, initial price

### Mathematics Programs:
- Compile with: `mpic++ filename.cpp -o output`
- Run with: `mpirun -np 4 ./output`
